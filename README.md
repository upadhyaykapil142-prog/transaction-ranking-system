# 🚀 Transaction Ranking System

A backend system that processes user transactions, generates user summaries, and maintains a fair ranking system based on transaction activity.

---

## 📌 Live Demo
- Frontend: https://transaction-ranking-system.vercel.app
- Backend API: https://transaction-ranking-api.onrender.com 

---

## 🧠 Problem Statement

This project simulates a real-world ranking system where users perform transactions, and the system:
- Stores transaction data safely
- Prevents duplicates
- Maintains consistency under concurrent requests
- Generates user rankings based on activity

---

## ⚙️ Features

- ✅ Create transaction (`POST /transaction`)
- 📊 Get user summary (`GET /summary/:userId`)
- 🏆 Get global ranking (`GET /ranking`)
- 🔒 Duplicate request handling
- ⚡ Fast API responses
- 📦 Clean backend architecture
